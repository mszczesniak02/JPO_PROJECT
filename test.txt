klasa File -> jest klasą macierzystą, odpowiedzialną za wpisanie zawartości pliku do vectora
            -> każdy obsługiwany rodzaj plików zawiera wspólne funkcje, któruch implementacja różni się

            -> potrzebny jest tylko setter, aby dowiedzieć się jaka jest lokalizacja (dir ) pliku

            -> statyczna zmienna zawiera ilosc otwartych (pozytywnie ) plikow


            -> funkcje ktorych implementacja rozni sie dla roznych rodzajow plikow

            -> wspołna funkcja do odczytu metadanych: praw dostępu, właściciela oraz innych zmiennych jest wspolna
                dla otwartego pliku 

            te funkcje to: 
                    -> analyze_most_frequent_word
                    -> analyze_functions
                    -> analyze variables
                    -> analyze comments
                    -> analyze empty lines
